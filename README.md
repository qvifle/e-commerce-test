# E-commerce test task

## Запуск проекта

### Docker

```bash
docker-compose up -d --build
```

Открываем в браузере [http://localhost:3000](http://localhost:3000)

### Локально

```bash
npm i
npm run build
npm run start
```

Открываем в браузере [http://localhost:3000](http://localhost:3000)

## Технологии

 - Next.js
 - TailWind
 - Zustand
 - TypeScript
 - React Query
